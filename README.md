# Series_Temporales_Daily_Minimum-_Temperatures-
En este proyecto se construyó un sistema de pronóstico para la temperatura mínima diaria de Melbourne utilizando ventanas deslizantes de 30 días y un horizonte de 7 días. Se compararon distintos enfoques (LSTM, XGBoost, regresión armónica con términos de Fourier y un ensamble Armónica + XGBoost) sobre la misma partición de entrenamiento y prueba.
